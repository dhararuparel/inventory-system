Download Android App
How to Install on Android
  1. Tap the green **Download Android App** button above directly from your device.
  2. Open the downloaded `.apk` file (via Notification or Files app).
  3. If prompted, enable **"Install unknown apps"** or **"Allow from this source"** for your browser in Android settings.
  4. Tap **Install**, launch the app, and log in with your dashboard account credentials.
Windows App

Gokul Desktop App

Installable desktop setup with direct Vercel engine integration and offline session persistence.

  • File Format Setup Installer (.exe)
  • Current Version v1.0.0
  • File Size ~65 MB
  • Last Updated June 8, 2026
Download Windows App
How to Install on Windows
  1. Click the blue **Download Windows App** button to get the setup installer.
  2. Double-click `GokulCycleTyresSetup.exe` to run the setup wizard.
  3. If Windows Defender SmartScreen shows a warning, click **"More Info"** and then select **"Run anyway"**.
  4. The app will automatically install and create a desktop launcher.
